Bundled Noto Fonts
==================

The font files in this directory are part of the Noto font family by Google:

- Noto Sans (Regular + Italic, variable weight/width)
- Noto Sans Mono (variable weight/width)
- Noto Sans Symbols 2 (Regular)
- Noto Emoji (variable weight)
